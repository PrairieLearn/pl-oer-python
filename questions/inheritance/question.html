Create a parent and a child class


Python classes offer a structured approach to programming, allowing us to encapsulate data and functionality together, making our code more modular, reusable, and easy to understand. We can create our own objects with its properties and methods using Python classes. Inheritance is a concept in object-oriented programming, where a new class (subclass/derived class) can inherit attributes and methods from an existing class (base class/parent class).

Part 1: Create a parent class named Company that has following properties / methods:

  • name (string): representing the name of the company.
  • head (string) representing the head of the company.
  • The Company parent class should also have a method / function called get_info() that returns a formatted string f"Company:{self.name}|Head:{self.head}".

Part 2: Create a child class named ITCompany that inherits from the Company parent class. The ITCompany class should introduce an additional attribute and method:

  • services (list): representing the services provided by the IT Company.
  • The ITCompany child class should also define a method / function called get_services that returns the list of services offered by the company

Part 3: You are given a company_info tuple containing a name string, head string, and services list information of a ITCompany. After you create your ITCompany class, define the ITCompany object it_company_obj using the company_info tuple.
