Creating a Student Class


Python classes offer a structured approach to programming, allowing us to encapsulate data and functionality together, making our code more modular, reusable, and easy to understand. We can create our own objects with its properties and methods using Python classes.

Below is a simple Person Class. A Person Object would have a name property, an age property, and an introduction() method.

class Person: def __init__(self, name, age): self.name = name self.age = age def introduction(self): print(f"My name is {self.name}. I am {self.age} years old.")

Part 1: Create a Student class that has a netid property, a major property, and an age property.

Part 2: You are given a student_info tuple containing a netid string, major string, and age string information of a student. After you create your Student class, define the Student object student_obj using the student_info tuple.

Some details:

  • The type of Student.netid should be a string.
  • The type of Student.major should be a string with all uppercase letters.
  • The type of Student.age should be an integer.
  • You do not need to create methods for your Student class.
  • The Python string upper() method may be helpful.
